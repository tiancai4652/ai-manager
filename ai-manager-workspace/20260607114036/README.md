# 奇偶判断工具

一个简单的 Node.js CLI 工具，输入数字即可判断奇偶性，支持连续多次判断。

## 功能特性

- 交互式命令行界面，通过 `readline` 接收用户输入
- 自动验证输入，非数字会提示重新输入
- 支持连续判断，无需重复启动程序
- 零依赖，仅需 Node.js 环境

## 安装步骤

```bash
git clone <仓库地址>
cd ai-manager-workspace-20260607114036
npm install
```

> 项目无第三方依赖，`npm install` 仅初始化环境。

## 使用方法

```bash
npm start
```

运行后交互示例：

```
请输入一个数字：42
42 是偶数
是否继续？(y 继续，其他退出)：y
请输入一个数字：7
7 是奇数
是否继续？(y 继续，其他退出)：n
感谢使用，再见！
```

输入非数字时会自动提示并重新询问：

```
请输入一个数字：abc
请输入有效的数字
请输入一个数字：
```
