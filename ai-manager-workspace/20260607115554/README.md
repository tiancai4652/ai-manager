# Hello World — Node.js 示例脚本

一个最简单的 Node.js 脚本，在控制台输出 `hello world`。

## 功能特性

- 控制台输出 hello world 文本
- 零依赖，无需安装任何第三方包
- 即开即用，适合 Node.js 入门学习

## 安装步骤

需要提前安装 [Node.js](https://nodejs.org/)（v12 及以上即可）。

```bash
# 克隆项目
git clone <your-repo-url>
cd <your-repo-dir>
```

无需 `npm install`，本项目没有任何外部依赖。

## 使用方法

直接用 Node.js 运行脚本：

```bash
node hello.js
```

输出：

```
hello world
```

## 项目结构

```
hello.js   # 主脚本，调用 console.log 输出 hello world
```
